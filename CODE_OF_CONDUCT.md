# Code of Conduct

This project follows a respectful and inclusive research collaboration policy.

## Our Standards

We expect all contributors to:

- Be respectful in discussions
- Provide constructive feedback
- Focus on technical and research improvement
- Avoid personal attacks

## Unacceptable Behavior

- Harassment or discrimination
- Personal insults
- Public disclosure of private information

## Enforcement

Maintainers may remove contributions that violate this policy.

This project aims to foster healthy academic collaboration.
